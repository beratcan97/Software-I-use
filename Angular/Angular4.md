# Angular 4+

## Commands

### Create a app
> ng new my-app

### Launch the server
> ng serve --open

### Create a component
> ng g component components/user

### Button
> <button (click)="addNumber()">{{clicked}}</button>

### Props
> {{myprop}}

### Don't update after submit
> return false;

### Push to first place in a array
> unshift(); instand of push

### List of a array
> <li *ngFor="let hobby of hobbys;">{{hobby}}</li>
